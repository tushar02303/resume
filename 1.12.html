Website Demo Widget

Embeddable widget to showcase live website demos on any webpage. Perfect for portfolio sites, agency websites, or demo showcases.

Live Demo Widget

How to Use This Widget

To embed this widget on your website, add the following code:

<!-- Include CSS -->
<link rel="stylesheet" href="widget.css">

<!-- Widget Container -->
<div id="demo-widget-container"></div>

<!-- Include JavaScript -->
<script src="widget.js"></script>

<!-- Initialize Widget -->
<script>
new DemoWidget({
containerId: 'demo-widget-container',
apiUrl: 'api.php',
theme: 'light', // or 'dark'
limit: 6,
showCategories: true,
showViews: true,
openInModal: true
});
</script>

Features

Easy Integration

Just include CSS and JS files, add a container div, and initialize the widget.

Responsive Design

Works perfectly on mobile, tablet, and desktop devices.

Theme Support

Switch between light and dark themes with one line of code.

Analytics

Tracks view counts for each demo to see what's popular.

API Endpoints

The PHP backend provides the following API endpoints:

  • GET api.php?action=get_demos - Get all demos
  • GET api.php?action=get_demo&id={id} - Get specific demo
  • POST api.php?action=increment_views - Increment view count
  • POST api.php?action=add_demo - Add new demo
